# forma-oagrinho2026
Repositório destinado a formação do NRE Ivaiporã 2026 agrinho
Este site funciona como uma interface de gestão agrícola. Ele demonstra como a tecnologia de sensores pode auxiliar o produtor do futuro a manter o equilíbrio ambiental, evitando desperdícios e garantindo que cada pé de morango receba exatamente o que precisa para crescer de forma sustentável.
